Anti-Hacker Lab - Arcilla, EJ D.
Week 11 Activity – Advanced Web Development

This project is a secure web application built using CodeIgniter 4.
It demonstrates the implementation of:

CSRF Protection
XSS Protection
Secure Form Handling
User Input Escaping

The interface is designed with a custom Looney Tunes-inspired theme for a creative and engaging UI.

Features

✅ CSRF Security using csrf_field()
✅ XSS Protection using esc()
✅ Secure POST Form Submission
✅ Styled User Interface
✅ CodeIgniter 4 Framework
✅ Custom Looney Tunes Theme

Technologies Used
PHP
CodeIgniter 4
HTML5
CSS3
Composer
